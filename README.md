A lightweight **Python weather bot** that fetches and displays **real-time weather** for a user-provided location using a public Weather API.

## Demo
- **Video demo:** [`Clima_Final.mp4`](./Clima_Final.mp4)

## Features
- Search weather by **city/location input**
- Displays key conditions like:
  - Temperature
  - Weather description (e.g., Clear, Clouds, Rain)
  - Humidity
  - Wind (if available from the API response)
- Simple, single-file runnable script: `Clima_Final.py`

## Tech Stack
- **Python 3**
- HTTP + JSON API calls (commonly via `requests`)
- Environment variables for API keys (recommended)
